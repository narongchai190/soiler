## Common Organic Fertilizers / ปุ๋ยอินทรีย์ทั่วไป

| Property | Value |
|----------|-------|
| N-P-K | 2-1-1 |
| Application Rate | 500-1,000 กก./ไร่ |
| Cost | 3-5 บาท/กก. |
| Benefits | ปรับโครงสร้างดิน, เพิ่มจุลินทรีย์ |

## Bio-fertilizers / ปุ๋ยชีวภาพ

### Rhizobium (ไรโซเบียม)
- ใช้กับถั่วเหลือง, ถั่วลิสง
- ตรึงไนโตรเจนจากอากาศ
- ลดการใช้ปุ๋ยไนโตรเจน 30-50%

### Mycorrhiza (ไมคอร์ไรซา)
- เพิ่มการดูดซับฟอสฟอรัส
- เพิ่มความทนแล้ง
- ใช้ได้กับพืชส่วนใหญ่

### Azolla (แหนแดง)
- ใช้ในนาข้าว
- ตรึงไนโตรเจน 30-60 กก./ไร่/ปี
- ควบคุมวัชพืชในนาข้าว

## Application Guidelines / แนวทางการใส่

### Timing / เวลาที่เหมาะสม
1. **ก่อนปลูก**: ใส่ปุ๋ยหมักหรือมูลสัตว์ 2-4 สัปดาห์ก่อนปลูก
2. **ระหว่างการเจริญเติบโต**: เสริมด้วยน้ำหมักชีวภาพ

### Application Methods / วิธีการใส่
- **หว่าน**: กระจายทั่วแปลงแล้วไถกลบ
- **ใส่หลุม**: ใส่ก้นหลุมก่อนปลูก
- **รดน้ำ**: ใช้น้ำหมักชีวภาพเจือจาง

## Organic vs Chemical Comparison / เปรียบเทียบอินทรีย์กับเคมี

| Aspect | Organic | Chemical |
|--------|---------|----------|
| Nutrient Release | ช้า | เร็ว |
| Soil Structure | ปรับปรุง | ไม่มีผล |
| Microorganisms | เพิ่ม | อาจลด |
| Cost/Nutrient | สูงกว่า | ต่ำกว่า |
| Long-term Effect | ดีกว่า | ชั่วคราว |

## Organic Certification / มาตรฐานอินทรีย์

### Thai Organic Standards
- ไม่ใช้ปุ๋ยเคมีสังเคราะห์
- ไม่ใช้สารเคมีกำจัดศัตรูพืช
- ระยะเปลี่ยนผ่าน: 2-3 ปี

### Cost-Benefit Analysis
- ค่าใช้จ่ายสูงขึ้น 20-30%
- ราคาผลผลิตสูงขึ้น 30-50%
- คืนทุนภายใน 2-3 ปี
